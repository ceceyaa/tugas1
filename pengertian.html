Pengertian HTML

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat dan mengatur konten pada halaman web. HTML adalah bahasa dasar untuk pembuatan halaman web, dan digunakan untuk merancang struktur dan tampilan konten pada sebuah situs web. HTML digunakan bersama dengan CSS (Cascading Style Sheets) dan JavaScript untuk membuat halaman web interaktif dan menarik.
