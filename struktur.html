Struktur HTML


Tag HTML

Tag 'HTML' adalah elemen-elemen markup yang digunakan dalam bahasa HTML untuk menggambarkan struktur dan konten dari halaman web. Setiap tag HTML dibungkus dalam tanda kurung sudut '<,>' dan dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut.Tag HTML adalah dasar dari pembuatan halaman web, dan mereka digunakan untuk merancang struktur dan tampilan halaman, serta memberikan makna pada konten. Kombinasi dari tag 'HTML' dengan atribut dan CSS membantu dalam menciptakan tampilan dan interaktivitas halaman web.

Tag Head

Tag head dalam HTML adalah bagian dari sebuah halaman web yang digunakan untuk menyediakan informasi metadata dan elemen-elemen yang terkait dengan pengaturan dan deskripsi halaman, tetapi tidak ditampilkan secara langsung kepada pengguna pada halaman web itu sendiri Tag head adalah bagian penting dari setiap halaman web karena mengandung informasi yang membantu mesin pencari, peramban, dan pengguna memahami halaman Anda dan bagaimana tampilannya harus diatur.

Tag Body

Tag 'body' dalam HTML digunakan untuk mengelompokkan dan menentukan konten utama yang akan ditampilkan pada halaman web. Ini adalah bagian dari elemen HTML yang menampilkan konten yang dapat dilihat dan diakses oleh pengguna ketika mereka mengunjungi halaman web.ag 'body' adalah bagian yang menampilkan konten aktual yang dilihat oleh pengguna ketika mereka mengunjungi halaman web Anda. Konten di dalam tag 'body' akan ditampilkan pada peramban web ketika halaman web diakses, sementara bagian lain, seperti metadata dan pengaturan, berada dalam tag 'head'.

Tag Footer

Tag 'footer' dalam HTML adalah elemen yang digunakan untuk menandai bagian bawah dari suatu elemen konten atau halaman web. Ini biasanya berisi informasi tambahan tentang konten atau halaman tersebut. Tag 'footer' adalah salah satu elemen semantik dalam HTML5 yang membantu dalam merancang tata letak dan struktur halaman web.tag 'footer' digunakan untuk mengelompokkan informasi hak cipta dan kontak di bagian bawah halaman web. Ini adalah tempat yang tepat untuk informasi tambahan yang relevan dengan konten halaman tersebut, seperti hak cipta, tautan ke halaman kebijakan privasi, informasi kontak, dan sebagainya. Penggunaan tag 'footer' membantu dalam meningkatkan struktur dan makna konten halaman web, serta membantu dalam pemahaman halaman web oleh mesin pencari dan perangkat pembaca layar.
